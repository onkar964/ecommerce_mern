This is e-commerce responsive website 

**Frontend**
React
React Router Dom

**Backend**
MongoDB
Express
Node
Mongoose

**Project Setup**
Clone entire project repo.
**Setting up client :**
npm install
npm start
**Starting up server :**
node ./index.js

**Admin**
for Adding products
npm run dev
